# ltdidong_baitap_01

## Nội dung:

1. Cài đặt môi trường làm việc với Android Studio  
2. Tạo Project Baitap01 gồm 01 Activity như sau:  
   - Hiển thị hình và thông tin của từng sinh viên  
   - Sử dụng `de.hdodenhof.circleimageview.CircleImageView` để bo tròn hình ảnh  
3. Ẩn thanh tiêu đề của Android  
4. Viết chương trình:  
   - Nhập 01 mảng `ArrayList` kiểu số  
   - In ra `Log.d` các số lẻ và số chẵn riêng biệt có trong mảng  
5. Viết chương trình:  
   - Nhập 01 chuỗi ký tự bất kỳ từ View (`EditText`)  
   - In ra View (`TextView`) và hiển thị Toast chuỗi đó đảo ngược và in hoa thông qua nút `Button`  
   - Ví dụ: `s="I LOVE YOU"` -> `s="YOU LOVE I"`
